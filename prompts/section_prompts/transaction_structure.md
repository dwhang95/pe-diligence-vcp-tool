# Prompt: Transaction Structure Analysis

Generate the Transaction Structure Analysis section for the PE ops diligence brief.

## Input Context
- Company: {company_name}
- Description: {description}
- Industry: {industry}
- EV range: {ev_range}
- Context notes: {context_notes}
- Research findings: {research_summary}

## Instructions

Analyze the proposed transaction structure from an operational and investment return perspective.
If deal structure details are not known, state what is typical for a company of this profile and flag what must be confirmed.

Cover:

**1. Investment Type**
Classify the investment type: primary (first institutional capital), secondary (buying from existing PE), minority, or control buyout.
State what this means operationally — e.g., primary deals often require standing up governance infrastructure from scratch; secondaries may have legacy systems or IC baggage.

**2. Proposed Deal Structure & Key Terms**
Note any known or likely structural features: rollover equity, management equity plan, earn-outs, reps & warranties, working capital peg, seller note.
Flag any terms that create operational risk or complicate value creation execution (e.g., earn-outs tied to revenue metrics create incentive misalignment if the ops thesis requires customer mix optimization).

**3. Minority Protections & Liquidity Rights (if applicable)**
If this is a minority investment, identify what protections are standard for this deal profile and what should be negotiated: board seat(s), information rights, anti-dilution, drag/tag along, ROFR, put/call mechanics.
Flag if the company's cap table structure (e.g., family-owned with multiple classes) could complicate these.

**4. Cap Table & Ownership Concentration**
What is the expected ownership structure post-close? Who retains equity and what are the retention incentives?
Flag key-man risk from a cap table perspective — if a single founder holds majority and rolls minimal equity, alignment risk is high.

**5. Key Dependencies for Return Realization**
What has to be true for this deal to generate target returns? Frame this as 3–5 explicit thesis pillars (e.g., EBITDA expansion through pricing, multiple arbitrage via sector consolidation, exit to strategic buyer).
Flag any pillar that is execution-dependent vs. market-dependent — execution-dependent risks are manageable; market-dependent risks are not.

## Format

Write as flowing analysis with bolded sub-headers for each of the 5 areas. Use crisp, directional language.
Flag unknown information explicitly: "Unknown — confirm in management meeting" or "Typical for this deal profile, but verify in LOI."
Length: 400–600 words.
